5G PHY Development for Tactical Communications
Project Overview

Led the development and optimization of critical 5G physical layer components at Octasic, focusing on performance enhancement for tactical communications applications.

Key Achievements
  • Achieved 2000x reduction in channel latency through strategic algorithm and SDR optimizations
  • Reduced memory usage of LDPC decoders by 20%
  • Led development of complete polar encoder/decoder chain
  • Enhanced data transmission speeds for both uplink and downlink channels
Technologies Used
SDR C++ DSP 5G NR MATLAB
Technical Components

Optimized critical 5G PHY components including:

  • Physical Downlink Control Channel (PDCCH)
  • Physical Broadcast Channel (PBCH)
  • Physical Uplink Control Channel (PUCCH)
  • LDPC Decoders with optimized memory usage
  • Polar encoder/decoder chain for enhanced transmission
Impact & Results

The optimizations resulted in:

  • Dramatically reduced channel latency (2000x improvement)
  • Enhanced memory efficiency in LDPC decoders
  • Improved data transmission speeds
  • More efficient embedded platform performance

Communications Project
Project Overview

A self-driven, open-source framework for simulating end-to-end digital communication systems, including modulation, error correction, OFDM, and performance analysis.

Key Features
  • Fully modular Transmitter/Receiver architecture
  • Supports BPSK, QPSK, and OFDM modulation
  • Implements Polar Code decoding (SC, SC-Flip)
  • Flexible channel modeling with noise and fading
  • Performance evaluation tools (BER, FER)
Technologies Used
Python Signal Processing OOP
Technical Details

Included Components:

  • Modulation & Demodulation (BPSK, QPSK, OFDM)
  • Error Correction Coding (Polar Codes, SC, SC-Flip)
  • Channel Models (AWGN, Rayleigh, Rician)
  • Performance Metrics & Visualization

This project is publicly available for use, modification, and contributions. Please refer to the GitHub repository for detailed documentation.

POCO: A Fast SC Polar Decoder
Project Overview

POCO is an open-source implementation of efficient polar decoders, designed for both research and practical applications in 5G NR systems.

Key Features
  • Support for multiple decoding algorithms (SC, SSC, Fast-SSC)
  • Configurable size parameters for enabled fast nodes
  • Fixed-point arithmetic support
  • Performance plotting capabilities
  • Various configuration files with citation information
Technologies Used
Python C++ MATLAB
Technical Details

Supported Algorithms:

  • Successive Cancellation (SC)
  • Simplified Successive Cancellation (SSC)
  • Multiple variants of Fast-SSC
