Machine Learning Competition for 5G
Project Overview

Enter an exciting exploration of cutting-edge communication technologies powered by 5G! With the advent of the fifth generation (5G) of mobile networks, the world has witnessed a revolutionary leap in wireless communication. 5G promises ultra-reliable low-latency communication, enhanced data rates, and massive connectivitytransforming how we interact, work, and innovate.

Why 5G is Exciting

5G is more than just faster download speeds. It's the backbone of modern innovations - Smart Cities, Autonomous Vehicles, AR/VR, Healthcare, and many more. To deliver on these promises, 5G networks rely on robust error correction mechanisms that ensure data integrity across noisy and unpredictable channels. Among these mechanisms, polar codes, invented by Prof. Erdal Arkan, have emerged as a game-changer.

Technologies Used
PyTorch TensorFlow Keras Kaggle 5G NR C++
Advancements in Error Correction for 5G

Error correction coding in 5G has undergone significant transformation. Unlike previous generations that relied on LDPC or Turbo codes for different layers of transmission, 5G employs polar codes for control channel transmissions due to their capacity-achieving performance and flexible design.

About This Dataset

This dataset contains features extracted from the LLR (Log-Likelihood Ratio) values of received polar-coded frames. It serves as a powerful resource for building machine learning models to predict decoding outcomes.

  • Features (llr_X): Derived from Log-Likelihood Ratios received over a simulated AWGN channel, these numerical values represent key characteristics of the received frame, capturing insights into channel conditions and potential decoding challenges.
  • Target (num_errors): The number of flips required during SC-Flip decoding, a metric critical for understanding decoding complexity and reliability.

The data is generated using a custom-built C++ polar decoder, refined during years of academic research, and is now made publicly available to foster innovation and collaboration in the 5G era.

Use Cases
  • Develop machine learning models for polar code error prediction
  • Analyze decoding complexity under varying channel conditions
  • Explore feature engineering techniques for decoding algorithms
  • Compare decoding strategies and evaluate their performance
Acknowledgments

This dataset builds on my doctoral research in polar codes and aims to inspire further advancements in 5G communication technologies. The dataset reflects real-world challenges and provides a unique opportunity to push the boundaries of decoding algorithms with machine learning.
